How to Update

Regularly update the installed npm packages to keep your Hyas website stable, usable, and secure.

On this page

  • Check for outdated packages
  • Update npm packages
  • Update Hugo
  • Clean install
  • Resources
💡
Learn more about semantic versioning and advanced range syntax.

Check for outdated packages#

npm outdated

Update npm packages#

Bump the versions in the devDependencies section of ./package.json.

Next, run:

npm update

Update Hugo#

In ./package.json, bump the Hugo version:

"otherDependencies": {
  "hugo": "0.91.2"
}

Next, run:

npm install

Clean install#

npm run clean:install

Next, run:

npm install
